# CSS Test EPUB

This is an EPUB generated with help from Claude to test out CSS parsing capabilities in CrossPoint Reader for XTEInk devices.

## Building

Clone this repo. Then, from the root directory, run:

```bash
zip -X0 css-test.epub css-test/mimetype && zip -rX9 css-test.epub css-test/META-INF css-test/OEBPS
```
